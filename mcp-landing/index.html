¿Qué es Model Context Protocol?

MCP es un protocolo estándar que permite a los modelos de lenguaje (LLMs) interactuar con herramientas y datos externos en tiempo real.

Conectividad

Conecta modelos de IA con herramientas, APIs y sistemas externos de forma estandarizada.

Herramientas

Permite a los modelos de IA utilizar herramientas como calculadoras, navegadores web, bases de datos y más.

Datos en tiempo real

Accede a información actualizada y recursos externos para mejorar las respuestas de los modelos.

Estándar abierto

Protocolo de código abierto que facilita la interoperabilidad entre diferentes modelos y herramientas.

¿Cómo funciona MCP?

MCP establece un protocolo de comunicación bidireccional entre modelos de IA y servidores de herramientas. El flujo básico es:

  1. El cliente MCP se conecta a un servidor MCP
  2. El modelo de IA solicita información sobre las herramientas disponibles
  3. El modelo decide qué herramienta utilizar y envía una solicitud
  4. El servidor MCP ejecuta la herramienta y devuelve los resultados
  5. El modelo incorpora los resultados en su respuesta final

Prueba MCP Localmente

Configura y ejecuta servidores MCP en tu entorno local para potenciar tus modelos de IA.

Requisitos previos

  • Node.js

    Versión 18.0.0 o superior

  • npm o yarn

    Gestor de paquetes para instalar dependencias

  • Python (opcional)

    Para servidores MCP basados en Python

  • Terminal o línea de comandos

    Para ejecutar los comandos de instalación

Instalación rápida

Instala los servidores MCP globalmente con npm:

npm install -g @upstash/context7-mcp @modelcontextprotocol/server-puppeteer @modelcontextprotocol/server-sequential-thinking @modelcontextprotocol/server-github

Nota: Para el servidor GitHub, necesitarás configurar un token de acceso personal.

Configuración para diferentes entornos

Para VS Code, crea o edita el archivo mcp.json en la carpeta de configuración de usuario:

{
  "servers": {
    "context7": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@upstash/context7-mcp@latest"]
    },
    "puppeteer": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-puppeteer"]
    },
    "sequential-thinking": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-sequential-thinking"]
    },
    "github": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "${GITHUB_TOKEN}"
      }
    }
  }
}

Ubicación en Windows: %APPDATA%\Code\User\mcp.json

Ejemplo de uso en código

Así es como puedes utilizar MCP en tu código para conectar un modelo de IA con herramientas externas:

import { Client } from "@modelcontextprotocol/sdk/client/index.js";
import { StdioClientTransport } from "@modelcontextprotocol/sdk/client/stdio.js";
import { Anthropic } from "@anthropic-ai/sdk";

// Inicializar cliente MCP
const transport = new StdioClientTransport({
  command: "npx",
  args: ["-y", "@upstash/context7-mcp@latest"],
});

const client = new Client(
  { name: "mcp-client", version: "1.0.0" },
  { capabilities: {} }
);

await client.connect(transport);

// Inicializar cliente Anthropic
const anthropic = new Anthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});

// Listar herramientas disponibles
const toolsResponse = await client.request(
  { method: "tools/list" },
  ListToolsResultSchema
);

// Crear mensaje con Claude usando las herramientas
const response = await anthropic.messages.create({
  model: "claude-3-5-sonnet-20241022",
  max_tokens: 1000,
  messages: [{ role: "user", content: "Busca documentación sobre React hooks" }],
  tools: toolsResponse.tools.map(tool => ({
    name: tool.name,
    description: tool.description,
    input_schema: tool.inputSchema
  }))
});
import asyncio
import os
from mcp.client import ClientSession
from mcp.client.stdio import stdio_client, StdioServerParameters
from anthropic import Anthropic

async def main():
    # Inicializar cliente MCP
    server_params = StdioServerParameters(
        command="npx",
        args=["-y", "@upstash/context7-mcp@latest"],
        env=None
    )

    async with stdio_client(server_params) as (stdio, write):
        async with ClientSession(stdio, write) as session:
            await session.initialize()

            # Inicializar cliente Anthropic
            anthropic = Anthropic(api_key=os.environ.get("ANTHROPIC_API_KEY"))

            # Listar herramientas disponibles
            response = await session.list_tools()
            tools = [{
                "name": tool.name,
                "description": tool.description,
                "input_schema": tool.inputSchema
            } for tool in response.tools]

            # Crear mensaje con Claude usando las herramientas
            response = anthropic.messages.create(
                model="claude-3-5-sonnet-20241022",
                max_tokens=1000,
                messages=[{"role": "user", "content": "Busca documentación sobre React hooks"}],
                tools=tools
            )

            print(response.content[0].text)

if __name__ == "__main__":
    asyncio.run(main())
